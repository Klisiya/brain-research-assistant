# Brain Research Tutor Project

## Project Overview

This is a Flask-based brain research learning website with an AI tutor.

## Project Rules

- All user interface text must be written in English.
- Do not mix Chinese and English in the interface.
- Preserve the existing visual style unless explicitly asked to redesign it.
- Keep the liquid-glass effects, particle background, animations, and brain-themed design consistent.
- Make focused changes and avoid modifying unrelated sections.
- Do not expose or commit API keys, passwords, or `.env` contents.
- Do not commit `.venv`, cache files, or generated temporary files.
- Review the existing project structure before adding new files.
- After making changes, run the application and check for errors.
- Explain which files were changed and summarize the result.
- Run the Flask application, open the local website in the browser, inspect the current design, and then make the requested change. Verify the final page visually before finishing.

## Project Structure

- Flask backend: `project/app.py`
- Templates: `project/templates/`
- Static files: `project/static/`
- Dependencies: `requirements.txt`
