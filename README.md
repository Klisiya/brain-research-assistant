# Brain Research Assistant

A Flask-based brain research learning assistant website with an AI tutor feature.

## Features

- Brain research themed learning website
- AI tutor interaction page
- English-only user interface
- Responsive layout
- 3D brain model display
- Loading state for AI responses

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Git / GitHub

## How to Run

```bash
cd project
python app.py
```
